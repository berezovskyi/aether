Aether VoID Statistics Tool

Aether is a tool for viewing and generating statistical information about RDF datasets in the form of VoID descriptions.

Given a SPARQL endpoint, the tool can generate an extended VoID description containing a wide variety of statistical spreads describing the dataset.

These can then be viewed in order to get a general overview of the dataset.

The viewer can also compare dataset descriptions to each other, so that for example changes in a dataset can be glanced.

Alternatively, the viewer can also be used in live mode, where it directly queries a SPARQL endpoint for the statistics it shows.
