Aether VoID Statistics Tool

"Content for your VoID"

Given a SPARQL endpoint, the aether tool can generate an extended VoID description containing a wide variety of statistical spreads describing the dataset.

These can then be viewed in order to get a general overview of the dataset.

The viewer can also compare dataset descriptions to each other, so that for example changes between two versions of a dataset can be glanced.

Alternatively, the viewer can also be used in live mode, where it directly queries a SPARQL endpoint for the statistics it shows.

Example Descriptions

  • DBPedia public SPARQL endpoint compared between versions 3.9 & 3.8: visualized / as data
  • Europeana: visualized / as data
  • Virtual International Authority File (VIAF) compared between 2014-02-15 & 2013-11-24: visualized / as data
  • Full international DBPedia 3.9: visualized / as data
  • Finnish edition of DBPedia 3.9: visualized / as data
